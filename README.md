### SpringBoot 项目meng说明

#### 开发支持
>- 开发工具: IDEA-2017
>- 数据库: Mysql-5.6.x(因为项目中使用了current_timestamp,5.5以下版本不支持)
>- 内存数据库: redis

~ 注意: 一下技术点并非全部列举, 而是选取了比较有代表性的技术点

#### 涉及技术点
>- 统一返回值处理
>- 统一异常处理
>- 权限过滤器
>- 日志过滤器
>- 分布式session处理(cookie+redis数据库实现)
>- 多redis数据库(当前已注掉,项目中需要时会开启)
>- swagger2 API文档实现
>- 数据合法性后台验证

#### 整合业务处理
>- email发送(当前未实现)
>- 阿里短信发送(当前未实现)
>- websocket实现web端通讯(当前未实现)
>- 结合WebNotifications实现实现浏览器弹出通知(当前未实现)
